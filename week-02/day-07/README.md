# Неделя 2 - День 7. Сохранение контекста

## Задание

**Добавьте агенту сохранение контекста:**
- храните историю диалога (messages) в JSON или SQLite
- при перезапуске агента загружайте историю обратно
- продолжайте диалог так, как будто агент не выключался

**Проверьте на практике:**
- начните диалог
- перезапустите приложение
- продолжите диалог и убедитесь, что агент помнит прошлые сообщения

**Результат:**
- Агент, который сохраняет и восстанавливает контекст между запусками

## Процесс проверки

### ЗАПУСК №1 — Создаём сессию «Персонаж для RPG»

```text
/new Персонаж для RPG
```

```text
Давай создадим персонажа. Его зовут Элларион, он эльф-волшебник, ему 247 лет. Его родной лес был сожжён драконом по имени Вирмигорт, и теперь Элларион ищет способ отомстить. Его главное оружие — посох из лунного дерева по имени "Селестия".
```

```text
У него есть верный спутник — говорящий филин по имени Бубон, который на самом деле замаскированный фей. А ещё Элларион коллекционирует старинные карты.
```

```text
/quit
```

### ЗАПУСК №2 — Проверяем память между запусками

```text
Напомни, кто мой персонаж и чего он хочет?
```

```text
Добавь ему магическое заклинание, которое он выучил в память о своём лесе.
```

### Создаём новую изолированную сессию

```text
/new Кулинарный помощник
```

```text
Ты помнишь про моего эльфа Эллариона?
```

### Переключаемся обратно на старую сессию

```text
/sessions
```

```text
/switch
```

```text
А теперь вспомни всё про Эллариона и его спутника Бубона.
```

### Проверяем очистку истории

```text
/reset
```

```text
Ты помнишь Эллариона?
```

# AI Advent Challenge

Это рабочий журнал курса-челленджа [AI Advent Challenge](https://mobiledeveloper.tech/ai_advent_challenge) #8, цель 
которого научиться быстро использовать инструменты ИИ. Здесь находится код заданий, конспекты лекций и прогресс 
прохождения.

## Правила челленджа

- Лекции с новой темой выходят в начале каждой недели.
- Задания выходят **пн–пт в 14:00 МСК**. Всего 7 недель.
- Формат сдачи каждого задания: **видео + код**, причём **видео важнее кода**.
- Результаты вношу в общую таблицу курса.
- **Не сдал 5 заданий за неделю → выбываю.**
- Опоздание по сдаче — максимум **15 минут**.
- Принцип курса: всё делаем **через ИИ, код руками не пишем**.

## WebUI

SPA-чат для работы с JarvisAgent через браузер.

**Запуск:**
```bash
python .\webui\app.py 
```

После запуска открыть в браузере: http://127.0.0.1:5000

**Веб-интерфейс поддерживает:**
- Чат с LLM через JarvisAgent
- Создание/переключение/удаление сессий
- Настройки модели, температуры, max_tokens, context_limit
- Слэш-команды: /help, /stats, /sessions, /clear и другие

**Требования:** Python 3.12+, зависимости из `requirements.txt`.

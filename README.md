# AI Advent Challenge

Страница: https://mobiledeveloper.tech/ai_advent_challenge

Интенсив, чтобы быстро научиться использовать инструменты ИИ.

Как выглядит процесс:
- Каждый день (кроме выходных) вы получаете очередное задание. Приступайте когда вам удобно и сдавайте когда хотите
- В начале каждого блока заданий вы получаете от меня видео с объяснением того как работает механизм, что он из себя представляет и что нужно понять по итогам блока
- Вы не конкуренты друг другу. Работайте вместе, помогайте выполнять задания и успешно осваивайте материал

## Структура проекта

```
ai-advent-challenge/
├── agents/          # JarvisAgent — основной агент
│   └── memory/      # SQLite-база (история сессий)
├── webui/           # Веб-интерфейс (Flask)
│   ├── app.py
│   ├── templates/
│   └── static/
├── docs/            # Конспекты лекций и прогресс
└── week-*/          # Материалы заданий по неделям
```

## WebUI

SPA-чат для работы с JarvisAgent через браузер.

**Запуск:**
```bash
python .\webui\app.py 
```

После запуска открыть в браузере: http://127.0.0.1:5000

**Веб-интерфейс поддерживает:**
- Чат с LLM через JarvisAgent
- Создание/переключение/удаление сессий
- Настройки модели, температуры, max_tokens, context_limit
- Слэш-команды: /help, /stats, /sessions, /clear и другие

**Требования:** Python 3.12+, зависимости из `requirements.txt`.
